About xslbook-inlines

xslbook-inlines is a XSL library which is one of parts of xslbook. This library includes XSLT match templates which displays commonly used HTML inline elements.

xslbook elements that this library provides are named with single character. The reason is that we want to match these elements with HTML elements possibly, and want to reduce amount of typing since these elements are written in texts frequently. There are xslbook elements which are not contained in HTML elements. These elements are changed CSS styles against these default styles of transformed HTML elements..

All elements provided by this library supports data operations with elements provided by xslbook-do library. All elements can display values of specified nodes in same or another XML file, and can use conditional branchings or repetitions.

Usage

This section explains how to import xslbook-inlines and call its templates in your XSL application or library.

Import xslbook-inlines

To import xslbook-inlines.xsl into another XSL file, the following handworks are needed. Since xslbook-inlines depends on some xslet libraries, they are needed to import, too.

  1. Add the namespace declarations of xslbook and xslutil in a XSL file.

    <xsl-stylesheet version="1.0"
      xmlns:xsl="http://www.w3.org/1999/XSL/Transform"
      xmlns:ut="https://github.com/xslet/2020/xslutil"
      xmlns:book="https://github.com/xslet/2020/xslbook"> <!-- This! -->
      ...

  2. Add a import element of xslbook-inlines in the XSL file.

      ...
      <!-- dependency XSL files are already imported in xslbook-inlines.xsl -->
      <xsl:import href="./xslbook-inlines.xsl"/>
      ...
